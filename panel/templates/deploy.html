{{ icon('cloud', 18) }} مرحله ۱ — Cloudflare Worker

خودکار

پنل با REST API کلودفلر فایل deploy/cloudflare-worker/worker.js را به‌صورت ES module آپلود می‌کند، زیردامنه‌ی workers.dev را پیدا و فعال می‌کند، و مقدار WORKER_URL داخل خود Worker را با آدرس واقعی‌اش جایگزین می‌کند.

{{ icon('key', 16) }}
توکن ندارید؟ این لینک صفحه‌ی ساخت توکن کلودفلر را باز می‌کند. ساده‌ترین راه: از فهرست قالب‌ها روی Use templateِ گزینه‌ی Edit Cloudflare Workers بزنید، بعد Continue to summary و Create Token.
{{ icon('link', 14) }}باز کردن صفحه‌ی توکن کلودفلر اگر دسترسی‌ها از قبل تیک نخورده بودند، دستی این را بدهید: Account · Workers Scripts · Edit. (اختیاری: Account Settings · Read تا Account ID هم خودکار پیدا شود.)
توکن فقط یک بار نمایش داده می‌شود — همان لحظه کپی و اینجا وارد کنید.
دسترسی لازم: Account → Workers Scripts → Edit. {% if token_saved %} یک توکن ذخیره‌شده موجود است؛ اگر این کادر را خالی بگذارید از همان استفاده می‌شود. حذف توکن ذخیره‌شده {% endif %}
